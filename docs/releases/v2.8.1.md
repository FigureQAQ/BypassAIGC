# BypassAIGC v2.8.1

这是一个配置诊断与失败任务恢复修复版本。

## 修复内容

- 全局 `OPENAI_API_KEY` 与 `OPENAI_BASE_URL` 现在会自动用于润色、增强和情感处理阶段。
- 示例占位 Key 会被识别为未配置，不再等到任务处理时才返回 `401`。
- DeepSeek 等 OpenAI 兼容接口不会再错误显示“Base URL 未配置”。
- 失败任务卡片会显示 API Key、Base URL、限流或超时等具体原因。
- 修改 `.env` 并重启后，可以点击“继续处理”从失败段落恢复。

## 已验证

- 后端 7 项自动化测试通过。
- 前端生产构建通过。
- DeepSeek API 真实请求成功。
- 5589 字 Word 文档的失败任务已成功恢复并持续推进。

## 使用建议

首次使用仍推荐运行 `setup.bat`，依次填写 API Key、Base URL 和模型名称，然后运行 `start.bat`。
