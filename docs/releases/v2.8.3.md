# BypassAIGC v2.8.3

本版本优化工作台布局、文档提交流程和用户可见术语。

## 界面更新

- 工作台默认进入文档上传，不再默认显示文本粘贴框。
- Word 文档作为首选提交类型，同时保留 PDF、Markdown 和短文本入口。
- 上传区域支持点击和拖拽，并明确说明 Word 保格式处理范围。
- 处理模式改为双列卡片，减少纵向占用：
  - 降低 AIGC 率
  - 降重
  - 降低 AIGC 率 + 降重
  - 自然表达

## 术语统一

- 欢迎页、工作台、管理后台、会话监控和配置页面统一使用新名称。
- Markdown 文档同步更新默认上传流程和双阶段处理说明。

## 验证

- 前端生产构建通过。
- 文档上传入口仍支持 `.docx`、`.pdf`、`.md` 和 `.markdown`。
