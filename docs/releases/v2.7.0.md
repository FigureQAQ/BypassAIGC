# BypassAIGC v2.7.0

发布日期：2026-07-20

本版本重点提升运行性能、长时间任务稳定性和工作台使用体验。

## 主要更新

- React 页面按路由懒加载，减少首屏业务脚本。
- Word 编译移至工作线程，不再阻塞 FastAPI 事件循环。
- 增加任务取消、服务关闭清理和过期任务定时回收。
- SSE 使用有界消息队列，避免慢连接造成内存持续增长。
- 后台页面自动降低轮询频率，回到前台后立即同步任务状态。
- 工作台增加停止任务、历史搜索、状态筛选、任务统计和工具快捷入口。
- 登录页与工作台采用全新的响应式界面设计。
- 更新前端依赖，生产依赖安全审计为 0 个已知漏洞。
- 新增 5 项后端运行时管理回归测试。

## 配置变更

新增可选配置：

```env
STREAM_QUEUE_MAX_SIZE=256
```

该配置限制单个 SSE 浏览器连接允许缓存的最大消息数量。

- FastAPI 健康检查：通过。
- Python 编译检查：通过。
