# BypassAIGC v2.8.0

发布日期：2026-07-20

本版本重点简化源码项目的配置和启动流程。用户只需设置一次 API，之后可直接双击启动。

## 三步使用

1. 双击 `setup.bat`，填写 API Key、Base URL 和模型名称。
2. 双击 `start.bat`。
3. 使用结束后双击 `stop.bat`。

## 主要更新

- 源码运行统一读取仓库根目录 `.env`。
- 删除重复配置模板，不再同步桌面或子目录 `.env`。
- 新增交互式 API 配置脚本，并自动生成管理员密码和 JWT 密钥。
- 自动生成本地访问密钥、创建本地用户并直接进入工作台。
- 首次启动自动创建根目录 `.venv` 并安装前后端依赖。
- 依赖未变化时跳过安装，显著缩短后续启动时间。
- 后端和前端均就绪后才提示启动成功。
- 默认仅监听 `127.0.0.1`，降低本地访问密钥暴露到局域网的风险。
- 使用 PID 文件和端口双重停止机制。
- 保留旧版 `package/start-app.bat` 作为兼容入口。

## 用户需要关注的文件

```text
setup.bat
start.bat
stop.bat
.env
.env.example
```

其余代码统一保存在 `package/` 目录，普通使用无需进入。
