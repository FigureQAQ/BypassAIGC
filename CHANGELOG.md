# 更新日志

本项目遵循语义化版本号，记录用户可见功能、性能和稳定性改动。

## v2.8.1 - 2026-07-20

### 修复

- 修复模型健康检查没有继承 `OPENAI_API_KEY` 和 `OPENAI_BASE_URL` 的问题。
- 修复 DeepSeek 等兼容接口已配置但仍显示 Base URL 未配置的问题。
- 增加占位 API Key 检测，避免任务启动后才因 `401` 失败。
- 失败任务卡片现在显示简洁、可操作的错误原因。

### 验证

- 新增模型配置回退与占位 Key 单元测试。
- 验证 DeepSeek API 鉴权及真实对话请求成功。
- 验证失败的 Word 文档任务可从断点恢复并持续推进。

## v2.8.0 - 2026-07-20

### 简化配置

- 源码模式统一读取仓库根目录 `.env`。
- 删除重复的 `package/.env.example`，根目录 `.env.example` 成为唯一模板。
- `.env.example` 顶部只展示 API Key、Base URL 和两个主要模型配置。
- `setup.bat` 支持交互式生成 `.env`、管理员密码和随机 JWT 密钥。
- 自动生成随机本地访问密钥，并在首次启动时创建无限使用次数的本地用户。

### 简化启动

- 新增根目录 `start.bat`，自动创建 Python 虚拟环境并按需安装依赖。
- 使用依赖文件 SHA-256 缓存，依赖未变化时跳过 `pip install` 和 `npm ci`。
- 启动后同时检查 FastAPI `/health` 和前端页面，不再提前报告成功。
- 浏览器自动打开本地访问链接并直接进入工作台，无需先创建卡密。
- 新增 PID 文件管理和 `stop.bat`，可可靠关闭尚未完成端口监听的进程。
- 旧版 `package/start-app.bat` 保留为兼容入口。

### 文档

- README 快速开始改为三步流程。
- 更新项目结构图、常见问题和 Release 使用说明。

## v2.7.0 - 2026-07-20

### 新增

- 工作台任务统计和专业工具快捷入口。
- 历史会话关键词搜索及全部、进行中、完成、失败状态筛选。
- 活跃优化任务一键停止。
- SSE 单连接消息队列容量配置 `STREAM_QUEUE_MAX_SIZE`。
- 后端运行时管理回归测试。

### 性能

- React 页面改为按路由懒加载，减少首次访问需要加载的业务脚本。
- Word 文档编译通过工作线程执行，避免阻塞异步事件循环。
- 页面进入后台后降低进度和队列轮询频率，回到前台时立即同步状态。
- SSE 队列达到容量上限时淘汰旧消息并保留最新状态，避免无界内存增长。

### 稳定性

- 取消并发等待时立即移除残留队列记录。
- 跟踪运行中的文档任务，支持取消和服务关闭时统一清理。
- 启动后台任务定时回收循环，清理超过保留期限的任务数据。
- 修复 Windows 非 UTF-8 控制台可能因任务日志符号导致异常的问题。

### 界面

- 重新设计登录页，增加产品能力介绍和响应式双栏布局。
- 重新设计工作台头部、统计卡片、历史面板和任务进度区域。
- 增加统一渐变背景、玻璃卡片和页面加载状态。

### 依赖

- 移除不应安装的第三方 `asyncio` 包。
- 更新前端锁定依赖；生产依赖安全审计无已知漏洞。

## v2.6.3 - 2026-05-20

- 改进文档上传、保格式导出、管理后台和本地启动体验。
- 增加 Windows、Linux 和 macOS 自动构建与 Release 分发。
